##爱你的100个漂流瓶

### 概述

- 表白专用。。
- 一天能捞起一个漂流瓶（随机的），漂流瓶内含有表白的话语，每个瓶子内的话都不同
- 一共好多好多个漂流瓶，如果集齐了所有，代表~~
- 还有几个模块
  - 收藏——可以看到收藏的表白漂流瓶
  - 总列表——可以看到所有已有的漂流瓶
    - 集齐后，可自动发送邮件给对方。
- 口令登录。可以给多个妹子使用，不同人不同口令。
- 后台管理。。。
